# Frontend Project Notes

This is a Next.js 16.2.3 project with React 19 and Tailwind 4.

## Key Files
- `src/app/page.tsx` - Home page with rankings
- `src/app/login/page.tsx` - Login with 2FA flow
- `src/app/vote/page.tsx` - Voting form (15 teams)
- `src/app/admin/page.tsx` - Admin dashboard
- `src/app/apply/page.tsx` - Voter application
- `src/app/pollsters/page.tsx` - Pollster listing
- `src/lib/api.ts` - API client

## Environment
- `NEXT_PUBLIC_API_URL` - Backend API URL
